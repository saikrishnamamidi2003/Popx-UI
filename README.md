
# PopX UI Assignment

This project is a pixel-perfect implementation of the PopX mobile UI design using React. The application replicates the provided design screens and demonstrates navigation between them.

## Live Demo

https://popx-ui-ochre.vercel.app/

## GitHub Repository

([Github =>](https://github.com/saikrishnamamidi2003/Popx-UI))

## Screens Implemented

* Welcome Page
* Create Account Page
* Account Settings Page

## Features

* Pixel-perfect UI based on the given design
* Mobile app interface centered on the webpage
* Page navigation using React Router
* Reusable component structure
* Clean and organized CSS styling
* Fully deployed using Vercel

## Tech Stack

* React.js
* Vite
* CSS
* React Router DOM

## Project Structure

src
components → reusable UI components
pages → application screens
assets → images and icons
styles → global CSS

## Installation (Run Locally)

Clone the repository

git clone https://github.com/saikrishnamamidi2003/Popx-UI

Navigate into the project

cd popx-ui

Install dependencies

npm install

Start development server

npm run dev

## Deployment

The project is deployed on **Vercel** for fast and reliable hosting.
